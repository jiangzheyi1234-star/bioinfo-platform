## 完成的功能

### ✅ 左侧面板：工具卡片列表
- 显示所有可用工具
- 搜索功能
- 工具计数
- 卡片选中状态
- 响应式网格布局

### ✅ 右侧面板：参数表单
- 工具信息展示（名称、ID、版本、分类）
- 输入文件选择（带浏览按钮）
- 参数配置（支持多种类型）
  - 文本输入
  - 数字输入（整数/浮点数）
  - 布尔选择
- 数据库路径配置
- 运行按钮
- 清空表单按钮

### ✅ Python ↔ JavaScript 通信
- `get_tools()` - 获取工具列表
- `get_tool_descriptor(tool_id)` - 获取工具详细信息
- `select_tool(tool_id)` - 选择工具
- `browse_file(input_id)` - 文件浏览对话框
- `run_tool(tool_id, params)` - 执行工具

## 使用方法

### 启动应用
```bash
# 方式1：批处理文件
启动完整版.bat

# 方式2：命令行
python -m ui.main
```

### 使用流程
1. 启动应用
2. 点击左侧导航栏的"病原检测"
3. 在左侧卡片列表中选择一个工具
4. 右侧会显示该工具的参数表单
5. 填写输入文件和参数
6. 点击"运行工具"按钮执行
